 # Installation
 - `git clone git@github.com:ionutradis/simpleshop.git`
 - `composer install`
 - `npm install && npm update`
 - `update database credentials in .env file`
 - `php artisan migrate:refresh --seed`
 - `php artisan serve`
 
 # COMMANDS
